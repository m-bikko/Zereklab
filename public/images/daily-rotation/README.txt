Папка для еженедельно ротирующихся изображений - ZerekLab
======================================================

❗ ВАЖНО: Эта папка больше НЕ используется для хранения изображений!

Новая система управления изображений недели:
1. Все изображения теперь хранятся в localStorage браузера
2. Управление происходит через админ-панель (/admin)
3. Поддерживаются форматы: JPG, JPEG, PNG (до 5MB)
4. Изображения автоматически меняются каждую неделю

Как использовать новую систему:
1. Войдите в админ-панель (/admin)
2. Перейдите на вкладку "Изображения недели"
3. Загрузите изображения через интерфейс
4. Система автоматически будет показывать разные изображения каждую неделю

Принцип работы:
- Изображения хранятся в localStorage с ключом 'zereklab_daily_images'
- Каждую неделю вычисляется индекс на основе номера недели в году
- Формула: неделя_года % количество_изображений
- Если нет изображений, показывается placeholder

Преимущества новой системы:
- Простое управление через админ-панель
- Не нужно загружать файлы на сервер
- Мгновенное обновление изображений
- Поддержка любого количества изображений
- Автоматическая оптимизация хранения

Для разработчиков:
- Изображения конвертируются в base64
- Структура данных: { id, name, data, uploadDate }
- Компонент: components/DailyImageRotation.tsx
- Админка: components/admin/DailyImageManagement.tsx
